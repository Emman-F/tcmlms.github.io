Backend Integration Guide

Connecting to PHP / Laravel or Node.js

All pages are built as frontend demos with fetch() / setTimeout() placeholders. Replace these with real API calls to your backend. Below are the key integration points.

// 1. LOGIN — tcm-lms-login.html const res = await fetch('/api/auth/login', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ id, password, role }) }); const { token, role } = await res.json(); window.location.href = routes[role]; // 2. SUBMIT ASSIGNMENT — student-dashboard.html const form = new FormData(); form.append('file', fileInput.files[0]); form.append('assignment_id', assignmentId); await fetch('/api/submissions', { method: 'POST', body: form }); // 3. ADD STUDENT TO CLASS — instructor-dashboard.html await fetch('/api/classes/{classId}/students', { method: 'POST', body: JSON.stringify({ name, student_id, email, section }) }); // 4. SAVE GRADE — instructor-dashboard.html await fetch('/api/grades', { method: 'POST', body: JSON.stringify({ student_id, subject_id, score, remarks }) }); // 5. BACKUP — admin-dashboard.html await fetch('/api/admin/backup', { method: 'POST' });
