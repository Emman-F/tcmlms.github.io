Course Description
This course introduces students to the fundamental concepts and technologies used in building modern web systems. Topics include HTML5, CSS3, JavaScript, responsive design principles, and an introduction to client-server architecture. Students will design and develop functional, accessible web pages and apply best practices in front-end development throughout the semester.
Learning Objectives
  • Apply semantic HTML5 structure to build accessible web pages
  • Use CSS3 (including Flexbox and Grid) to create responsive layouts
  • Write JavaScript to add interactivity and DOM manipulation
  • Understand the basics of client-server communication and APIs
  • Design and deploy a complete personal portfolio website
